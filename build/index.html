Грай відповідально з FAVBET.

Скарбничка — функціонал для контролю витрат, що дозволяє тимчасово обмежити частину коштів на твоєму балансі, зробивши їх недоступними для гри.

Як скористатися скарбничкою?

Переконайся:

Ти верифікований гравець.

У тебе є кошти на основному балансі.

Встановлюй ліміт:

  • Відкрий
  • Особистий кабінет
  • Баланс
  • Скарбничка

Вкажи суму для обмеження витрат. Вона буде недоступною для гри.

Повертай кошти:

  • У розділі
  • Скарбничка
  • натисни
  • Зняти

Обери суму, з якої хочеш зняти обмеження та повернути на основний баланс. Кошти стануть доступні для гри.
